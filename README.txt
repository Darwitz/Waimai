简易外卖前端演示
=================

说明：
- 这是一个纯静态的前端站点（HTML/CSS/JS），数据保存在浏览器 localStorage。
- 适合直接托管为静态网站（例如：Vercel、Netlify、GitHub Pages、又拍云等）。
- 不包含真实支付功能；如果需要接入微信支付/Stripe/PayPal，请让我帮你生成后端示例。

如何部署为可访问网址（推荐）：

1) GitHub + Vercel
- 在本地解压并初始化 git 仓库，push 到 GitHub。
- 在 Vercel.com 使用 "New Project" 选择该仓库，直接部署（Framework: Other / Static Site）。
- 部署完成后会得到一个可访问的网址，手机访问该网址即可使用。

2) Netlify
- 登录 Netlify，创建新站点，拖拽目录或连接 GitHub，部署即可。

3) GitHub Pages
- 创建 repo，把文件 push 到 gh-pages 分支或设置为 GitHub Pages 源，网址一般为 https://<用户名>.github.io/<仓库名>/
